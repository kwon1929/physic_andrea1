# 🚀 빠른 시작 가이드

## 자비스/아트레이디스와 대화하기

### 1. 환경 설정

```bash
source venv/bin/activate
```

### 2. 아트레이디스 실행

```bash
python atreides.py
```

### 3. 이름 선택
- **1번 (자비스)** 추천 - 짧고 인식 쉬움
- 2번 (아트레이디스) - 듄 팬용
- 3번 (커스텀) - 원하는 이름

### 4. TTS 선택
- **2번 (macOS 내장)** 추천 - 무료, 한국어 지원

### 5. 사용 방법

**대기 중:**
```
👂 대기 중... (말씀하세요)
```

**깨우기:**
```
당신: "자비스" (또는 "아트레이디스")
로봇: "네, 듣고 있습니다."
```

**명령하기:**
```
🎤 5초 동안 명령을 들을게요...
당신: "빨간 블록을 집어줘"
로봇: "네, 빨간 블록을 찾아서 집을게요."
[동작 실행]
로봇: "완료했습니다."
```

**다시 대기 모드로 자동 복귀**

## 💡 예시 대화

```
당신: "아트레이디스"

```

## 🎯 지원하는 웨이크워드

### 자비스 모드 (추천)
- **자비스** ⭐ - 가장 인식 쉬움
- **Jarvis**
- **제비스**

### 아트레이디스 모드
- **아트레이디스**
- **Atreides**
- **아트레이데스**
- **아트레** (짧게)

### 유연한 매칭
- 부분 단어도 인식 ("자비", "비스" 등)
- 띄어쓰기 무시
- 발음 변형 자동 감지

## ⚡ 팁

1. **조용한 환경**: 배경 소음이 적을수록 인식률 향상
2. **명확한 발음**: 또박또박 말하기
3. **적당한 거리**: 마이크에서 30-50cm
4. **침묵 감지**: 말이 끝나면 자동으로 처리 시작

## 🛠️ 문제 해결

**웨이크워드가 감지 안 됨:**
- 더 크게 말하기
- "아-트-레-이-디-스" 천천히 발음
- 영어로 "Atreides" 시도

**음성 인식 실패:**
- API 키 확인 (.env 파일)
- 인터넷 연결 확인

**종료:**
- `Ctrl+C` 누르기
